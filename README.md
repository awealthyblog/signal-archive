# Signal archive (timestamped commitments)

Ogni segnale live viene committato come hash SHA-256 (cartella `commitments/`) nel momento in cui e' generato. Il file completo viene pubblicato in `delayed/` dopo 45 giorni. Chiunque puo' verificare che l'hash del file delayed coincida con il commitment pubblicato in anticipo.

Questo prova che il track record non e' stato riscritto a posteriori.
